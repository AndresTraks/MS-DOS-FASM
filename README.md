# MS-DOS v1.25, v2.0, v4.0 Source Code for FASM
This repo contains the source code for MS-DOS v1.25, v2.0 and v4.00, modified to be assembled with [flat assembler](https://flatassembler.net/).

# Assembly
```
cd v1.25
.\compile.bat
```
